# Frontend

## Purpose
Owns the client application UX, static assets, and frontend build pipeline.

## Local development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start a local dev server:
   ```bash
   npm run dev
   ```

## Build
Create a production build:
```bash
npm run build
```

## Run (production preview)
Serve the built artifacts locally:
```bash
npm run start
```
